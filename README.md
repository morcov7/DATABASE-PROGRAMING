## 1. 사용기술

MVC2 구조를 기반으로 Servlet JSP를 이용해 개발했습니다.

- 언어 : JSP, Javascript, JSON, HTML5, CSS, Bootstrap
- 개발 환경 : Eclipse, oracle/SQL developer, Tomcat, ERwin



## 2. 담당 업무

- 각각 모두 직접 mvc 모든 과정을 개발
- 이동선 
  1. Club  (동아리)
  2. Customer (회원)
  3. list, detail 디자인 템플릿
